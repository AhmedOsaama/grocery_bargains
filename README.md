# Grocery Bargains

A grocery app combining various supermarkets with price comparisons and todo lists.

Skills and concepts:
- Firebase
- RESTful APIs
- Third party SDK integration(Mixpanel, Branch, Sentry etc.)
- Real-time chat
- Local storage
- Authentication
- Cloud functions
- Push notifications
- Deep linking
- CI/CD with Codemagic
- State management
- Multi-language
- In-app purchases
